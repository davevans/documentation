# Temporal CLI activity command reference

Activity commands operate on [Activity Executions](/activity-execution).

Activity commands follow this syntax:
`temporal activity [command] [command options]`

## complete

The `temporal activity complete` command completes an [Activity Execution](/activity-execution).
Along with completing the Activity, the result given upon return can be set as well.

Example:

```
temporal activity complete --activity-id=MyActivity --result=ActivityComplete
```

Use the following options to change the behavior of this command.

<Tabs>
<TabItem value="flags" label="Flags">

| Flag | Type | Description |
| --- | --- | --- |
| [--activity-id](/cli/cmd-options#activity-id) | string | Activity ID. |
| [--identity](/cli/cmd-options#identity) | string | Identity of the user submitting this request. |
| [--result](/cli/cmd-options#result) | string | JSON result to return. |

</TabItem>
<TabItem value="global-flags" label="Global flags">

| Flag | Type | Description |
| --- | --- | --- |
| [--address](/cli/cmd-options#address) | string | Temporal Service gRPC endpoint. |
| [--codec-auth](/cli/cmd-options#codec-auth) | string | Authorization header for codec server requests. |
| [--codec-endpoint](/cli/cmd-options#codec-endpoint) | string | Remote codec server endpoint. |
| [--color](/cli/cmd-options#color) | string-enum | Output coloring option. |
| [--command-timeout](/cli/cmd-options#command-timeout) | duration | Command execution timeout. |
| [--env](/cli/cmd-options#env) | string | Active environment name. |
| [--grpc-meta](/cli/cmd-options#grpc-meta) | string[] | HTTP headers for requests. |
| [--namespace](/cli/cmd-options#namespace) | string | Temporal Service Namespace. |
| [--run-id](/cli/cmd-options#run-id) | string | Workflow Run ID. |
| [--tls](/cli/cmd-options#tls) | bool | Enable base TLS encryption. |
| [--tls-ca-path](/cli/cmd-options#tls-ca-path) | string | Path to CA certificate. |
| [--tls-cert-path](/cli/cmd-options#tls-cert-path) | string | Path to x509 certificate. |
| [--tls-disable-host-verification](/cli/cmd-options#tls-disable-host-verification) | bool | Disable TLS hostname verification. |
| [--tls-key-path](/cli/cmd-options#tls-key-path) | string | Path to private key. |
| [--tls-server-name](/cli/cmd-options#tls-server-name) | string | Override target TLS server name. |
| [--workflow-id](/cli/cmd-options#workflow-id) | string | Workflow ID. |

</TabItem>
</Tabs>
